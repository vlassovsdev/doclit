FAQ

Частые вопросы

Мои файлы в безопасности?

Да. Файлы обрабатываются на изолированном сервере и удаляются сразу после загрузки результата. Мы не храним и не анализируем содержимое ваших документов.

Какие языки поддерживает OCR?

Tesseract 5 поддерживает 100+ языков включая русский, казахский, английский, немецкий, арабский. Многоязычные документы распознаются автоматически.

Нужна ли регистрация?

Нет — первые 3 конвертации в день доступны без регистрации. Для 5 файлов/день на бесплатном тарифе нужен только email.

Есть ли API для интеграции?

Да, REST API доступен на тарифе Business. Документация с примерами на Python, JS, curl. Webhook-нотификации о готовности файлов.

Как работает AI улучшение?

Для изображений — апскейл через Real-ESRGAN, удаление шума, коррекция перекоса. Для текста — Claude AI для рерайта, суммаризации, перевода.

Можно ли развернуть у себя?

Self-hosted версия в планах. Напишите нам — для enterprise клиентов готовы обсудить on-premise деплой с вашей инфраструктурой.
